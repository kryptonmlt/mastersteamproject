To open project use eclipse
The application needs 6 arguments to run:
1) path to input data file
2) theta value in float
3) M representing query limit
4) K number of clusters in online kmeans for query clustering
5) alpha which is the learning rate in online kmeans for query clustering
6) L file which contains subspaces in format x,y,z,...,width etc...

NOTE:
in DIST folder you can find an example of the application :)
windows: run qge.bat
mac/linux: run qge.sh
